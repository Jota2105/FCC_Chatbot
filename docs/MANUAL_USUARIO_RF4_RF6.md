# Manual de Usuario – RF4, RF5 y RF6

## 1. Reportes de Historial IA (RF4)
### Acceso
1. Ingrese al sistema con rol **admin**.
2. Abra el menú **Administración → Asistente IA → Historial de Consultas**.

### Uso
- Filtre por **ID Usuario**, **Session ID**, texto o fechas.
- Revise la tabla de resultados.
- Presione **Descargar CSV** para exportar un reporte.

## 2. Capacitaciones (RF5)
### Acceso
1. Ingrese al sistema con rol **admin**.
2. Abra el menú **Administración → Capacitaciones**.

### Uso
- **Nueva Capacitación**: ingrese los campos obligatorios (Título y Fecha Inicio).
- **Editar**: seleccione una capacitación y ajuste sus datos.
- **Eliminar**: confirme la eliminación.

## 3. Asesoramiento Virtual (RF6)
### Acceso
1. Ingrese al sistema con rol **admin**.
2. Abra **Asistente IA → Asesoramiento Virtual**.

### Uso
- Escriba su consulta sobre procesos o trámites.
- Reciba respuesta del asesor virtual.
- Use **Regresar** para volver al panel anterior.
